## 📌 Project Overview

This project was completed as part of the Digital Literacy course. The main objective of this project is to understand and apply essential digital skills such as creating online profiles, using digital tools, communicating professionally, and staying safe online.

Through this project, I explored different platforms and learned how to use them effectively for academic and professional growth.

---

## 🧩 Task 1: Digital Literacy Infographic

In this task, I created an infographic using Canva to spread awareness about digital literacy. The infographic includes topics such as the meaning of digital literacy, useful digital tools, safe internet practices, professional online presence, and email etiquette.

This task helped me understand how to present information in a simple and visually appealing way.

📁 Folder: task-1-presentation/

---

## 🌐 Task 2: Digital Portfolio

In this task, I created my digital presence by setting up profiles on GitHub, LinkedIn, and Kaggle. These platforms are useful for showcasing skills, building a professional network, and practicing technical concepts.

I also created a GitHub profile README to introduce myself and my learning goals.

📁 Folder: task-2-portfolio/

## 📧 Task 4: Email Etiquette & Communication

In this task, I wrote two professional emails—one requesting an assignment extension and another applying for an internship. I also created a checklist of do’s and don’ts for responsible social media use.

In this task, I studied UPI fraud as a common cybercrime. I created a case study explaining how such frauds happen and also prepared a prevention checklist to help students stay safe online.

This task increased my awareness about online security and safe digital practices.
